Detailed Description

Class responsible for operations that can be performed on a position.

Position represents a data type of a set of coordinates in x-y space that can be manipulated. Basic numerical operations (such as addition, multiplication, etc.) can be performed on Position objects. These can be performed one one or both axes. More complex operations are also defined, such as addition in a specific direction. Position works in an x-y coordinate space where right corresponds to the positive x direction and down corresponds to the positive y direction
