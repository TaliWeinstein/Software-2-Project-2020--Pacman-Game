Detailed Description

The Maze environement, a derived class of Drawable.

This class keeps track of all the mazeEntities with which the Dog can interact, enables the maze to be updated when collisions occur and enables the maze to be reset in the case of a Game restart. Maze implements the virtual function given in the Drawable abstract base class enabling the Maze to be drawn by Screen. The Maze object also is able to inform clients whether a section is forbidden or whether a new position is valid based on whether the desired location is a mazeEntities::WALL or a mazeEntities::LOCKEDSECTION. The Maze class contains functionality to convert from a row- column Maze object coordinate representation to that of an x-y Screen coordinate.
