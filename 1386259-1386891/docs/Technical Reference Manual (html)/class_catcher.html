Detailed Description

Class modelling the ghost implementation from the original pacman.

The Catcher class is a game object which upon collsion with the Dog character, will reduce the number of lives the Dog has. The Catcher has two different movement modes, a smart mode for following the Dog and a random mode whereby it picks a movement direction at random. The Catcher class's main responsibilities are to update its position, retrive its postition, provide its representation and a manner for respawning. The Catcher is a subtype of Character which in turn is a subtype of Drawable. The Catcher subsequently implements the virtual functions of both these abstract base classes.

◆ update()

void Catcher::update ( Position  dog_pos = {0,0})

Updates the Catcher's position during movement.

This function detemines how to update the Catcher's position based on the movementMode enabled. In the case of movementMode::smart, the position is updated with reference to the Dog's position based on the fastest path to reach the Dog. When movementMode::random is enabled, the Catcher arbitrarily updates its position using a random number generator.

Parameters
dog_posThecurrent position of the Dog, utilised for movementMode::smart. This is specified in x-y coordinate space.
