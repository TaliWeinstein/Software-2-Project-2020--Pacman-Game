Detailed Description

Class modelling the user controled Super Pac-Man from the original game.

The Dog class is a game object controlled by the user to move around the Maze. The Dog has class is a subtype of Character which in turn is a subtype of Drawable. The Dog subsequently implements the virtual functions of both these abstract base classes. Additionally, the Dog is able to update itself and change its direction.
