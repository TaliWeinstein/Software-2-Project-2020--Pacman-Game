Detailed Description

Class responsible for getting user input.

UserInput takes an SFML event and converts it into an enumeration, Pressed. The events are retrieved from an SFML RenderWindow object, and these events are limited to the arrow keys, spacebar, enter key, and window closed. Repeated key events are ignored.

UserInput::Pressed UserInput::input ( )

Retrieves the most recent user input.

The most recent user input is found and converted to an enumeration of type Pressed. If no event has occurred or an event not dealt with has occurred then the Pressed::NONE is returned.

Returns
An enumeration of type Pressed corresponding to the event that has occurred
