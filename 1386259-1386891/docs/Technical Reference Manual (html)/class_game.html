Detailed Description

Enables the Game to be run and function.

This class accounts for the logic of running a game of Dog-Catcher by enabling the game to be constructed, run and ended.

void Game::run ( )

Responsible for running the Game object once started.

This function runs the Game in accordance to the prompts from the user as well as from collisions which have occured. This running of the Game corresponds to updating game objects and subsequently displaying the objects with the Screen class.
