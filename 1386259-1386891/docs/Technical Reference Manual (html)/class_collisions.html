Detailed Description

Keeps track of collisions between the Dog and other Game entities.

The Collisions Class enables all the collisions between Dog and the various mazeEntities or Catchers to be recorded. The Collisions class is able to store multiple collisions which occur between the Dog and other game entities such as a collision with mazeEntities::BONE and a Catcher. The Collisions class provides the necessary information for Game when assessing with what the Dog collided and how the Game should subsequently respond.
