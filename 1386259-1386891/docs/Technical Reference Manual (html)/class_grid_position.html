Detailed Description

Class responsible for operations that can be performed on a grid position.

GridPosition represents a data type of a set of coordinates in row-column space that can be manipulated. Basic numerical operations (such as addition, multiplication, etc.) can be performed on GridPosition objects. These can be performed one one or both axes. More complex operations are also defined, such as addition in a specific direction. GridPosition works in a row-column coordinate space where down corresponds to the positive row direction and right corresponds to the positive column direction
