Detailed Description

Class used to model the movement of the Characters within the allowed space.

This class is utilised by both the Catcher and Dog objects in order to navigate the maze environment. Thi class therefore, captures the rules inherant in the maze regarding allowed sections.

◆ restartMovement()

void Movement::restartMovement ( )

Enables the movement of a Character to be restarted.

Movement is restarted in certain instances such as after the game is paused and after the game is restarted by the user. This restart occurs by resetting the clock of the characters.
