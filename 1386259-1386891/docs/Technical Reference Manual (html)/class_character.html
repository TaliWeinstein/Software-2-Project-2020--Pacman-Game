Detailed Description

Abstract base class modelling all movable Drawable Entities.

The Character class is a subtype of the abstract base class Drawable. The Character class is used to model the Dog and the Catcher, the two entities within the game which are capable of movement.

◆ unpauseCharacter()

virtual void Character::unpauseCharacter ( )
pure virtual

Virtual function for unpausing the Characters after a user triggered pause.

This virtual function is used to set the contract that all Characters must be able to unpause themselves after a user triggered pause and subsequent unpause. This ensures that Character movement is unaffected and unaltered between the GameActive and GamePaused game states.

Implemented in Catcher, and Dog.
