Detailed Description

Finds the best suited direction in which the Catcher should move.

CatcherMovement is a class which specifies the direction in which the Catcher can move depending on whether Catcher::movementMode::random or Catcher::movementMode::smart is set. When Catcher::movementMode::random is set, a random direction from all allowed directions is chosen via a random number generator. When Catcher::movementMode::smart is set the Catcher's movement is dictated by the direction which will enable the Catcher to collide with the Dog the fastest.

Determines the best direction for Catcher to collide with Dog.

This function uses pathfinding tools to find the best direction for the Catcher to move in based off of the dog's position. The function considers all the possible directions that Catcher can move in. It then considers the length of the path between Catcher and the dog by doing a breadth-first search to find the path between the two. The best direction to move in is the one with the shortest path

Chooses an allowed direction randomly for movement.

This function uses a random number generator to choose an arbitrary allowed direction in which to move the Catcher. This movement is only enabled when Catcher::movementMode::random is set. The direction is chosen such that the Catcher will not get stuck anywhere in the maze, but may move aimlessly away from the Dog.
