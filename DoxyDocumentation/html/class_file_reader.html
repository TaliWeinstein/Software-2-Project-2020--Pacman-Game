Detailed Description

Enables the reading of a maze into a Maze object from a file.

This class enables the a specified text maze file to be read into the Maze. The file must conform to the size of 32 rows by 18 columns. Additionally the appropriate symbols must be utilised to represent mazeEntities. These symbols are defined by the order of the mazeEntities with a 0 representing the mazeEntities::WALL while a 7 represent the mazeEntities::STEAK. There can only be one mazeEntities::START_DOG and one mazeEntities::START_CATCHER
